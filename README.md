# gcode-ripper
Cad viewer. Can read gcode data from `.ngc .txt .gcode` files and export the content int dxf format. 
Supported gcode commands : 

  `G0` 
  `G1` 
  `G2` 
  `G3`
  
Dxf output : 

  `3d lines` 
  `2d arc's`
  `2d circles`
